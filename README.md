# Validation Tools
Validation tools scripts

## Dependencies

- arosics
- GDAL
- numpy

## Installing via Git

```
python3 -m pip install git+https://github.com/fronzag/validation_tools.py
```

or

```
git clone https://github.com/fronzag/validation_tools
cd validation_tools
pip install .
```
